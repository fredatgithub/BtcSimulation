# Btc Simulation
# 
This application checks for BTC rates in Euro and in Dollar

## Description

BtcSimulation est une application qui permet de suivre en temps réel le cours du Bitcoin (BTC) dans deux devises :
- **Euro (EUR)**
- **Dollar américain (USD)**

L'application récupère les taux de change actuels et permet de simuler la valeur d'un montant donné en BTC selon les devises sélectionnées.
